Myblog

个人博客后台管理

Myblog是基于GPL协议的真正开源的PHP博客系统,功能强大,操作简单，界面清爽.(后台建议使用Chrome或者火狐浏览器，以获得最佳体验)。Myblog具有以下特点:

  • 缓存机制

    使用Memcache和文件缓存加快页面访问速度，减少数据库开销。

  • 搜索机制

    使用Sphinx作为搜索引擎，大大加强了搜索功能的效果。

  • 灵活的插件和模版机制

    灵活的插件和模版机制,使得扩展和定制变得简单。
